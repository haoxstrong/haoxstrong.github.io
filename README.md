## 🚀 快速开始
### 本地开发（可选）
如果你想在本地预览博客效果：

```bash
# 克隆仓库
git clone https://github.com/haoxstrong/haoxstrong.github.io.git
cd haoxstrong.github.io

# 安装依赖
bundle install

# 本地预览（访问 http://localhost:4000）
bundle exec jekyll serve
